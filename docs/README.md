# CineForge AI Pro — Docs

This folder contains all project documentation.

## Contents

| File / Folder | Purpose |
|---------------|---------|
| `bob-log/` | Screenshots and logs from IBM Bob (Ask/Plan/Code) sessions |
| `CHARTER.md` | Project vision, goals, success criteria, scope, and team roles |
| `MEMORY.md` | Living memory — key decisions, current status, important context |
| `AGENT.md` | Rules for AI agents and Copilot working in this repo |
| `ARCHITECTURE.md` | High-level system design and component responsibilities |
| `SECURITY.md` | Encryption, watermarking, and data handling rules |
| `ROADMAP.md` | Development roadmap linked to the July 2026 deadline |

## IBM Bob Log Convention

Save Bob session screenshots to `bob-log/` using this naming format:
```
YYYY-MM-DD_<short-topic>.png
```
Example: `2026-07-08_script-parser-planning.png`

After each session, add an entry to the table in `MEMORY.md` under "IBM Bob Usage Log".
